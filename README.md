# Activity Tracker Calendar

A simple Python desktop application to track your daily activities with an interactive calendar interface.

## Description

Activity Tracker Calendar helps you organize and monitor your daily habits like exercise, reading, and meditation. The app features an intuitive calendar view, customizable categories, and visual statistics to help you stay on track with your goals.

## Getting Started

### Prerequisites

- Python 3.7 or higher
- pip (Python package manager)


### Installation

1. Clone the repository
```bash
git clone https://github.com/Radji-radji/activity_tracker.git
cd activity-tracker-calendar
```

2. Install required libraries
```bash
pip install tkinter tkcalendar matplotlib numpy
```


### Running the Application

```bash
python main.py
```


## Usage

1. **Select a date** - Click on any date in the calendar
2. **Add an activity** - Fill in the form with title, category, duration, and optional notes
3. **View statistics** - Click the statistics button to see your activity breakdown
4. **Manage categories** - Create custom categories with your preferred colors

## Technologies Used

- **Python** - Core programming language
- **Tkinter** - GUI framework
- **tkcalendar** - Calendar widget
- **Matplotlib** - Chart visualization
- **NumPy** - Statistical calculations
